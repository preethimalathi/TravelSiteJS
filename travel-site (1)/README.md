# Travel Site (Static Demo)

Open `index.html` in your browser to run the site locally.

Structure:
- index.html, about.html, destinations.html, booking.html, contact.html
- assets/css/styles.css
- assets/js/main.js, assets/js/sw.js
- assets/data/destinations.json

Features:
- Bootstrap 5 + custom CSS, responsive layout
- Destinations loaded from local JSON and client-side filters
- Booking prefill via query param or localStorage
- Simple multi-step booking page and contact form (client-side only)
- Accessible focus styles and lazy images
